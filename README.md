This project is a data conversion utility such as
- binary to hex
- hex to binary
- hex to decimal
- decimal to hex
- decimal to binary
- binary to decimal
- ascii to hex
- hex to ascii
- binary to ascii
- ascii to binary

the list will grow as per the requirement

the ui will have,
navbar with logo and title
left sidebar with menu items
dark mode / light mode toggle
mobile responsive design
react-icons for icons

default screen will have list of all the conversion options with search bar
